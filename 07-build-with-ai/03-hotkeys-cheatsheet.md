# ⌨️ Hotkeys Cheatsheet — Integrated Into Your Workflow

> Don't learn hotkeys in isolation. Learn them at the moment you need them.

The shortcuts below are grouped by **what you're trying to do**, not by tool — so you reach for the right key at the right time.

---

## 🏆 The 10 Hotkeys to Learn First

Start here. Master these 10 before anything else.

1. **Ctrl+D** — Duplicate element *(fastest way to copy-paste a shape)*
2. **Ctrl+Z** — Undo *(obviously — your safety net)*
3. **Ctrl+Shift+H** — Fit to screen *(reset your view when you're lost)*
4. **Ctrl+A** then **Arrow keys** — Select all + nudge everything
5. **Escape** — Deselect / exit edit mode *(gets you out of any stuck state)*
6. **R / E / T / A** — Tool shortcuts in Excalidraw and Figma *(rectangle, ellipse, text, arrow)*
7. **F2** (Draw.io) / **Double-click** — Edit text in a shape
8. **Ctrl+G** — Group shapes *(move them as one)*
9. **Ctrl+S** — Save *(build the habit)*
10. **Ctrl+E** (Draw.io) / **Ctrl+Shift+E** (Excalidraw) — Export

## 🎯 Workflow Hotkey Moments

Use this as a reference for **when** to reach for each shortcut:

**You're doing this** | **Reach for**
--- | ---
Got lost on the canvas | Ctrl+Shift+H (fit to screen)
Want a copy of that shape | Ctrl+D (duplicate in place)
Made a mistake | Ctrl+Z (undo)
Need to type inside a shape | Double-click or F2 (Draw.io) / Enter (Excalidraw)
Done editing text in a shape | Escape
Want to move multiple things | Ctrl+A then drag (or arrow keys to nudge)
Need to align shapes | Figma alignment shortcuts / Draw.io align panel
Ready to export | Ctrl+Shift+E (Excalidraw) / Ctrl+Shift+X (Draw.io)
Want to insert Mermaid | Arrange → Insert → Mermaid (Draw.io)
Want to preview Markdown | Ctrl+Shift+V (VS Code)
